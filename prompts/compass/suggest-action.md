## 提案ルール

1. **時間制約を厳守する**: 提案するタスクは必ず{{available_time}}分以内で完了できるものにする
2. **エネルギーレベルに合わせる**:
   - レベル1〜2: 受動的な活動（聴く、軽く読む、眺めるなど）
   - レベル3: 中程度の活動（調べる、メモする、簡単な作業など）
   - レベル4〜5: 能動的・集中が必要な活動（作る、書く、練習するなど）
3. **具体性を徹底する**: 曖昧な提案は禁止
   - 悪い例: 「英語を勉強する」
   - 良い例: 「英語の技術記事を1つ読んで、知らなかった単語を3つメモする」
4. **1つだけ提案する**: 選択肢を増やさない（多すぎる選択肢は行動を妨げる）

```json
{
  "suggestedTask": "具体的なタスク名",
  "reason": "なぜこのタスクを提案するかの説明",
  "timeEstimate": 25
}
```

- `suggestedTask`: 今日取り組む具体的なタスク名（行動が明確にわかる表現）
